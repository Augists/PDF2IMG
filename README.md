# PDF2IMG

Convert your **PDF** files into **images** online

Here is a [Demo](http://137.116.167.187:7219)

## Have a try

```bash
git clone https://github.com/Augists/PDF2IMG.git
cd PDF2IMG
go run main.go
```

Server will run at `http://localhost:7219`

## Why use the port *7219*

```text
P - 7
2 - 2
I - 1
G - 9
```
